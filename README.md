# FAL-KE
Formatunabhängige Allgemeine Lernumgebung: Katalogisieren erlernen

Für das Erlernen der Katalogisierung nach RDA stehen (zukünftigen) Bibliothekar*innen diverse Möglichkeiten zur Verfügung, wie einschlägige Lehrbücher oder diverse Internetressourcen (Schulungsunterlagen) der einzelnen Verbünde.
Schulungen für Katalogisierende und Unterricht für Studierende sind Methoden, die das Regelwerk praxisorientiert vermitteln. 

Nach der Erlernung der Theorie fehlen jedoch oft Übungsaufgaben, die selbstständig durchgeführt werden können und die über die Beispiele aus oben genannten Lehrbüchern hinausgehen. 
Gerade jedoch Übungen und Wiederholungen sind der Schlüssel für einen sicheren Umgang mit RDA und reduzieren somit letztendlich auch die Fehlerquote in der Katalogisierung.   

Die Projektidee entstand aus einer Übungsdatei der UB Regensburg, anhand derer das Katalogisieren mit RDA erprobt werden kann. Die Übungsaufgaben sind hierbei leider stark begrenzt, die Übungsumgebung imitiert auf Grund der Arbeit mit einer Eingabemaske jedoch die „echte“ Katalogisierung in einem entsprechenden Programm. Durch eine automatische Korrektur können die Aufgaben selbstständig gelöst werden und die Übenden werden auf Fehler hingewiesen.

Das Ziel dieses Projekts soll eine webbasierte Übungsplattform mit Feedbackfunktion zum selbständigen Üben der Katalogisierung nach RDA sein. Dabei soll eine umfangreiche, erweiterbare Datenbank hinterlegt werden, um einen möglichst vielfältigen Pool an Übungsaufgaben anbieten zu können.
